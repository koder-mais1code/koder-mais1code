Koder.Mais1Code

É uma plataforma de oportunização para jovens/adultos para desenvolver habilidades técnicas que os preparem para a entrada no mercado de trabalho brasileiro.

O maior objetivo aqui é desenvolver uma intuição computacional de base que possibilite a preparação para:

  • Empreender digitalmente, compreendendo as ferramentas de criação de websites e aplicativos;
  • Estar apto a prestar serviços de programação como freelancer ou numa vaga de trabalho na posição de júnior.

Proposta

  • Uso de tecnologias acessíveis para demonstrar práticas que podemos executar juntos e assim adquirir as primeiras percepções dos mecanismos de algoritmos. Proponho ambientes de programação com nenhuma instalação de recursos extras, como acontece em ambientes 100% online interativos.
    • Como promover? Através de uso do jsfiddle ou scrimba. Acredito que seja um ponto de partida muito simples para se adquirir as primeiras intuições computacionais para o dia-a-dia.
  • Preparação para vagas de fullstack júnior. As habilidades de um fullstack júnior possibilitam empreender, trabalhar como freelancer e trabalhar numa grande diversidade de atividades de equipes de programação profissionais, sejam fazendo apps e até mesmo projetando negócios digitais.
    • Como promover? Através de práticas de codificação de tecnologias de frontend, backend e de forma avançada Cloud e DevOps. Aqui eu irei iniciar lições práticas de Tooling para criação de ambientes básicos de frontend local com vscode, remoto com Facebook CodeSandbox. Backend é possível ser feito com o Glitch (dos criadores do Trello e iniciativa da Black Girls Code por Kimberly Bryant).

  • Ninguém é uma caixa vazia. Temos histórias e experimentações pessoais anteriores. Quero conhecer mais de perto a vida da pessoa e mostrar que pensar em algoritmos é apenas saber dividir bem alguma tarefa e organizar da forma mais simples e direta possível para criar novas experiências de se ver o mundo de hoje.
    • Como promover? Através de um primeiro bate-papo para eu conhecer a história de vida da pessoa e ela conhecer um pouco da minha. Depois eu começo uma explicação sobre os caminhos possíveis da computação na vida dela, a jornada que ela pode trilhar. Chamo isso de First Meeting.
  • Evolução da prática com montagem de portfólio pessoal. A pessoa vai ser instruída a criar seu profile em redes sociais de programadores como GitHub e publicar repositórios com códigos e projetos desenvolvidos. Utilizar também o Google Sites para montar páginas pessoais com o conteúdo criado para chamar atenção de possíveis recrutadores.
  • Colocação em grupos da Rocket Seat. A Rocket Seat tem materiais e treinamentos avançados excelentes e grupos de redes sociais que pode alavancar a carreira de um fullstack. O foco de formação profissional deles é Full Stack.

Currículo Fullstack

  • Stack, Tecnologias e Linguagens propostas:
    • Frontend:
      • Essential Web Design com HTML, CSS e Javascript from Scratch (Vanilla JS to Browsers). Dialeto Javascript mais essencial para fazer funcionar em qualquer lugar (ES3). Montagem de layout de landing pages e páginas mais comuns da web com Twitter Bootstrap.
      • Functional & Object Javascript. Dialeto mais moderno do Javascript (ES2015). É necessário para se codificar em React e React Native e exigido ao se trabalhar com mobile.
      • Facebook React & React Native. Criação de apps com Javascript para mobile. Dialeto Javascript ES2015. Publicação de apps com Expo.
      • Advanced Concepts: Layout e UX com Figma.
    • Backend:
      • NodeJS: criação de APIs REST from Scratch e em Express.js. Dialeto Javascript mais moderno (ES2015). Conceitos de Cliente/Servidor e hospedagem de código. Um fullstack precisa saber rodar um servidor HTTP básico para fazer um app consultar um banco de dados, por exemplo.
      • SQL: compreensão de SQL para criação de bancos de dados e manutenção. A maioria das empresas não tem condições de manter profissionais exclusivos de bancos de dados e pedem um conhecimento em SQL como diferencial.
      • Google Firebase: criação de backend útil para criação de painéis de dados em tempo real, como em rastreamento de doenças como a covid-19.
    • Advanced Concepts
      • Cloud/DevOps (Hacking Skills):
        • Tecnologias de hospedagem de código em Cloud como Google Cloud e AWS com uso da AWS EC2, S3, Lambda, além de Google Cloud Functions.
